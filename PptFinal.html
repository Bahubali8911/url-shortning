After · The Agentic Era

One Upload. AI Handles the Rest.

CCS team uploads an Excel file with auditor request details. The entire KYC preparation pipeline runs automatically. HiTL review before delivery.

📊
Upload Excel via GUI
CCS uploads auditor request Excel with client IDs, review type, request ID. Human involvement ends here.
USER TRIGGER
📥
Agent Fetches from CMiO / DocDirect
Ingestion Agent searches & downloads all docs per client to GCS temp bucket. Saves master data summary.
DOCUMENT INGESTION AGENT
🔍
dbTextract OCR + Gemini Classifies
dbTextract extracts text → Gemini classifies document type, page range, confidence score. Creates document_segment entities.
CLASSIFICATION AGENT · AI DECISION ★
✅
Checklist Matching
Matches document_segments against KYC checklist items. Returns Found / NotFound / Uncertain per requirement.
CHECKLIST MATCHING AGENT
💾
Persist, Rename & Structure
Moves docs to permanent GCS: ClientId/Req_FKN_type1_type2.pdf. Deletes temp. Pub/Sub: completion.
PERSISTENCE AGENT
👁 Human in the Loop (HiTL)
Validator reviews prepared file package in GUI. Downloads ZIP, validates before sending to auditor. No results sent without human sign-off.
📁
Structured Output
AI-classified, correctly named, structured ZIP archive ready for HiTL review and auditor delivery.
